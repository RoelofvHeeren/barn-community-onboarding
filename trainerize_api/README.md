# Trainerize API Documentation

This directory contains comprehensive documentation for the Trainerize API, gathered for the Barn Community Onboarding project.

## Overview
The Trainerize API allows for management of clients, workouts, nutrition, and more. It is available to Studio and Enterprise customers.

- **Base URL:** `https://api.trainerize.com/v03/`
- **Rate Limit:** 1000 requests per minute per Group API token.
