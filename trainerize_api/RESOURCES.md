# API Resources

The Trainerize API exposes several resource categories. All endpoints abide by the base URL `https://api.trainerize.com/v03/`.

## Core Resources
- **Clients (User):** Manage client profiles and data.
- **Trainer:** Access trainer profiles.
- **Location:** Manage business locations.
- **UserGroup:** Group management.
- **UserTag:** Tagging system for users.

## Activity & Health
- **BodyStats:** Weight, measurements, etc.
- **HealthData:** General health metrics.
- **DailyCardio:** Cardio tracking.
- **DailyNutrition:** Macro and calorie tracking.
- **DailyWorkout:** Workout logs.
- **Habits:** Habit tracking.
- **Photos:** Progress photos.

## Scheduling & Communication
- **Appointment:** Booking and scheduling.
- **Calendar:** Calendar events.
- **Message:** In-app messaging.
- **UserNotification:** Push notifications.

## Programming
- **WorkoutDef:** Workout definitions.
- **WorkoutTemplate:** Reusable templates.
- **TrainingPlan:** Long-term plans.
- **Program:** Program assignment.
- **MealPlan:** Nutrition plans.

## Commerce & Engagement
- **Payments:** Payment transactions.
- **AppStore:** Add-ons.
- **Accomplishment:** Achievements/badges.
- **Challenge:** Challenge participants and tracking.
- **Compliance:** Adherence tracking.
